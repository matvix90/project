Level 1 - SQL Injection

SQL injection is a web security vulnerability that allows an attacker to interfere with the queries that an application makes to its database. The application doesn't implement any defenses against SQL injection attacks so you can try to access the site through the login form using sql query, for example OR 1=1.

Level 2 - Bruteforce

A bruteforce attack is a hacking method that uses trial and error to crack passwords, login credentials, and encryption keys. The application does not implement any defense against bruteforce attacks, so you can try to guess the pin code to enter. If you need help you can see into helped.py into app code.
